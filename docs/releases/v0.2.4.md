# remediation-core v0.2.4

## Changes

- Adds compact `manual_reviews[]` diagnostics to `result.json`.
- Reports rejected candidate details for manual remediation cases.
- Makes `NEEDS_MANUAL_REVIEW` exit non-zero in the CLI so pipelines do not look green when no remediation PR can be created.

## Compatibility

Use this release with `security-workflows v1.0.1` or newer for production remediation gates.
